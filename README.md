# XML Reader
## Descripción
App en node.js que procesa un XML de plantas y lo muestra en una página.
## Ejecución
En el terminal:
~~~
npm install
~~~
~~~
npm start
~~~
En el navegador: 
~~~
localhost:3000/xml
~~~
Introducir la URL del siguiente XMl:
https://www.w3schools.com/xml/plant_catalog.xml
